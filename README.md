# Meet Gamit's Portfolio - A

This is the source code for my personal portfolio website. It’s designed to bridge the gap between frontend development and motion design, showcasing clean code alongside a premium, cinematic visual experience. 

It features a modern dark aesthetic, glassmorphism UI cards, ambient background particles, and smooth scroll animations.

---

## Tech Stack & Libraries

* **Frontend:** Semantic HTML5, Custom CSS3, and Vanilla JavaScript.
* **Animations:** [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) for scroll triggers.
* **Icons & Typography:** Font Awesome v6.5.1 and Google Fonts (Inter).

---

## Key Features

* **Glassmorphism Design:** Premium translucent cards with sleek borders and backdrops.
* **Custom Interactive Cursor:** A smooth, custom-built cursor tracker that follows the user's mouse.
* **Ambient VFX:** Animated "Aurora" background glows and floating background particles to mimic a motion graphics timeline feel.
* **Dark Mode Toggle:** A built-in theme switcher integrated right into the navigation header.
* **Fully Responsive:** Smooth transitions across mobile, tablet, and desktop views.
